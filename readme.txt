Chartify is a powerful **WordPress Chart Builder Plugin** that will help you to create charts easily and quickly.

### Wordpress Chartify plugin

* [Chartify Homepage](https://ays-pro.com/wordpress/chart-builder)
* [Chartify Demo](https://ays-demo.com/chart-builder-demo/)


== Description ==

[Chartify](https://ays-pro.com/wordpress/chart-builder) is aimed to help you display dynamic and static results right on your website. 

Create amazing charts both manually and by transferring your data from Google Sheet account or query your preferred database. 

Visual charts can help you to create engaging content for your audience. No matter what profession you have, responsive charts are essential tools for you. 

Create an unlimited number of charts using the various chart types included in the plugin. 


== Types ==

*   Line Chart
*   Pie Chart
*   Bar Chart
*   Column Chart

== Features ==

*   Google Integration
*   Manual Data Entry
*   Database Entry
*   Live preview
*   Responsive design
*   Permissions of charts

== Creating a chart  ==

To insert the Chart into a Page, Post or text widget, copy shortcode [ays_chart id="chart_id"] and paste it at the desired place in the editor.

> <strong>Chart shortcode</strong><br>
> <strong>[ays_chart id="chart_id"]</strong>
